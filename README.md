# Парсер для сайта Ашан

## 1. Зависимости
- Python 3.11
- Git

## 2. Установка и Запуск
```shell script
# Создайте виртуальное окружение:
py -3.11 -m venv venv
# Активируйте виртуальное окружение:
venv\Scripts\activate.bat
# Установите зависимости:
pip install -r requirements.txt
# Запустите скрипт:
python main.py
```
